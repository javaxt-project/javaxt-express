This directory contains JavaXT javascripts.


WebSite.js is not a symbolic link! Do not delete!



Carousel.js is a symbolic links that were created using mklink. Example:

mklink /H Carousel.js "C:\Users\petya\Documents\JavaScript\javaxt-dhtml\src\carousel\Carousel.js"
mklink /H PageLoader.js "C:\Users\petya\Documents\JavaScript\javaxt-dhtml\src\utils\PageLoader.js"

mklink /H Effects.js "C:\Users\petya\Documents\JavaScript\javaxt-dhtml\src\utils\Effects.js"


NOTE: mklink cannot be used to delete a symbolic link. 
To remove a symbolic link, simply delete them as if you're removing a normal file. 
Just make sure you don't delete the original file.
