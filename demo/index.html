Welcome!

Express is a simple web server built using JavaXT. It is intended to help developers to get a web server up and running quickly. If you're reading this page, you're well on your way!

This demo has several cool features:

  • Dynamically generated content
  • Sliding navigation via a carousel control
  • URL rewriting via HTML5 push/pop state
  • Fail-safe fallback for browsers that don't have javascript enabled
  • Automated versioning of js and css files for proper caching

How to customize this site?

Now that you have your server up, you are ready to start adding content. Individual web pages are assembled using html and text files found in the web directory. These files contain html snippets which are then wrapped in a template found in the "style" folder.

You can start by opening one of the html files in the web directory (e.g. index.html) and edit the content. You do not have to restart the server to see the changes. Instead, you can simply refresh the browser. Don't worry, the server is smart enough to send the correct caching directives to the client.

You can also start playing with the layout and style of your website by editing the "template.html" and "style.css" files.

Finally, you can add and remove tabs by updating the tabs.txt file. Note that the tab file has a <%=Path> tag. This tag will be replaced with a relative path to the root directory. This is handy when hosting/supporting subsites. There are many similar tags. You can learn more here.
